First Time Windows Setup:
-Install MYSQL Server https://dev.mysql.com/downloads/installer/
-Default username is root, default password is empty unless set during installation
-Setup enviornment variable to mysql bin
-open command prompt
-cd into ../StateMinimumTracker/DBScripts
-enter the following command "mysql -u root -p (your password here)"
-it may require you to renter password
-enter the following command "source initDB.sql"
